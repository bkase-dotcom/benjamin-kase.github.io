Flow Field

A real-time generative system built with P5.js — curl noise and vortex dynamics driving ~8,000 particles, with color mapped to radius of curvature. Performance optimized with flat coordinate arrays and sub-stepping.

The System

A 2D vector field computed from curl noise determines particle trajectories. Vortex points inject rotational energy, creating emergent flow structures. Each particle's color is derived from its local radius of curvature — tight turns produce warm colors, gentle arcs produce cool tones — making the field's mathematical structure directly visible.

Curl Noise

Divergence-free noise field ensures smooth, continuous flow without sources or sinks. Particles follow streamlines that never converge to a point, preventing clustering artifacts.

Vortex System

Dynamic vortex points with configurable strength and falloff radius. Vortices can be added, moved, and decayed over time, creating evolving flow topologies.

Color Mapping

Each particle computes its instantaneous radius of curvature from consecutive position samples. This scalar value maps to a color palette — making the invisible geometry of the flow field perceptually legible.

Multiple Palettes

Switchable color palettes allow different aesthetic treatments of the same underlying data. Each palette is designed to preserve perceptual ordering so curvature relationships remain readable.

Performance

Rendering ~8,000 particles at 60fps requires careful optimization. Particle positions are stored in flat Float32Arrays rather than object arrays, eliminating garbage collection pressure. Physics simulation uses sub-stepping — multiple small integration steps per frame — to maintain stability at high particle velocities without reducing visual frame rate.

What It Means

Flow Field is about making mathematical structure visible and beautiful. The curl noise field is an invisible spatial function — the particles and their colors are the interface that makes it legible. Color-as-curvature transforms an abstract differential quantity into something you can see and intuitively understand. It's the same design challenge as SmartStretch or the Weighted Keyboard: physical (or mathematical) phenomena translated into a perceptible signal.
